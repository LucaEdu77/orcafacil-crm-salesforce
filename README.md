# OrçaFácil CRM - Salesforce

Projeto desenvolvido para prática de Apex e construção de um ecossistema dentro do Salesforce.

A ideia do projeto é simular um ambiente real de gestão de clientes, orçamentos e automações, aplicando lógica de negócio na prática para reforçar os estudos na plataforma.

## Objetivo

O OrçaFácil CRM nasceu como um projeto de aprendizado para evoluir conhecimentos em:

- Apex
- SOQL
- DML
- Triggers
- Flow Builder
- Automação de processos
- Modelagem de dados
- Integrações

Embora muitas funcionalidades possam ser feitas de forma declarativa com Flow Builder, a proposta é aprofundar a lógica de programação e entender melhor o backend do Salesforce.
